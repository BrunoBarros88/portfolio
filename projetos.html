Home do site de cadastro de receitas com sistema de menu, login, crud completo,paginação e forms.Na produção foi utilizado vs code studio, django, rest framework django, html5, css e um pouco de json e javascript.

Visualização das receitas na home, é possivel ver a descrição da receita,tempo de preparo, porções, categoria da receita, user criador e data de criação.

Dashboard de usuario onde é possivel ver receitas criadas, edita-las,exclui-las e fazer logout.

Sistema de cadastro de produtos com o uso da linguagem python, mySQL e Qt Designer onde é possivel cadastrar produtos de acordo com uma determinada categoria para manter registro e organização de produtos ligando diretamente o banco de dados com os dados fornecidos pelo sistema de cadastro.

Listagem dos dados ja inseridos no banco de dados e funcionalidades para editar e excluir algum dado e converter a lista para pdf.
